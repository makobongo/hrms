## Getting started with solutech crud operations
1.clone the project locally i.e git clone git@gitlab.com:makobongo/hrms.git

2.run composer install to install all the dependecies

3.run npm install to install necessary packages

4.create .env file and link to your database

5.migrate tables i.e php artisan migrate

6.install passport i.e. php artisan passport:install

## Configurations

2.Upon successful login create suppliers first, the button for this is available on the header of the table.
